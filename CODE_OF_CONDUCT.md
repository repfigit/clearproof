# Code of Conduct

## Principles

We write code to protect privacy. We expect the same respect for people as we demand for data.

- **Ship, don't posture.** Code and reasoned arguments win. Credentials and titles don't.
- **Disagree openly, resolve quickly.** Technical debates are healthy. Personal attacks are not. Attack the idea, never the person.
- **Default to transparency.** Decisions should be legible. Share context generously.
- **Respect the operator.** This software touches regulated systems and real money. Treat contributions with the seriousness they deserve.

## Unacceptable Behavior

- Harassment, threats, or doxxing
- Publishing private information without consent
- Sustained disruption of project work or discussions
- Conduct you wouldn't want attributed to you on-chain

## Enforcement

Report issues to **conduct@clearproof.dev**. Reports are handled discreetly and taken seriously.
